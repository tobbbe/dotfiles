# Agent Commands

User-invoked slash commands (`/something`) for Claude Code and opencode.
These live under `agents/commands/`.

Copied to `~/.claude/skills/` (Claude Code) and `~/.config/opencode/commands/` (opencode) by `reload`.

For auto-triggered skills, use `agents/skills/` instead.

## Adding a command

1. Create `agents/commands/<name>/SKILL.md`
2. Run `reload`
